# Horizon

An open-source collection of high-quality AI prompts for creators, developers, designers, and entrepreneurs.

## Features

- 1000+ AI Prompts
- ChatGPT Prompts
- Claude Prompts
- Midjourney Prompts
- Canva AI Prompts
- Image Generation Prompts
- Marketing Prompts
- Business Prompts
- Copy & Paste Ready
- Free and Open Source
